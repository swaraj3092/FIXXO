# 🏠 Smart Hostel Complaint Management System

> WhatsApp-based complaint system with AI classification, automatic routing, and real-time tracking

## 🎯 Problem Statement

**Who:** 10,000+ university hostel students
**Problem:** No efficient way to report and track maintenance issues
**Current Pain:** Manual registers, lost WhatsApp messages, 3-5 day resolution times

**Our Solution:** 
- Students report via WhatsApp (no app needed)
- AI auto-classifies and routes to correct department
- Email notifications with one-click resolution
- Full database tracking and accountability

**Impact:** Reduce resolution time from 5 days → <24 hours

## ✨ Features

- 📱 WhatsApp-native interface (Twilio API)
- 🤖 AI-powered classification (8 categories)
- ⚡ Priority detection (URGENT/HIGH/MEDIUM/LOW)
- 📧 Auto-email to departments
- 💾 PostgreSQL database tracking
- ✅ One-click resolution system

## 🛠️ Tech Stack

**Backend:** Flask (Python)
**AI/ML:** Rule-based classifier with ML upgrade path
**Database:** PostgreSQL (Supabase)
**Messaging:** Twilio WhatsApp API
**Email:** Resend API
**Deployment:** Render
