# Project Roadmap

## ✅ v1.0 - Hackathon MVP (Current)

**Features:**
- WhatsApp complaint submission
- Rule-based AI classification (8 categories)
- Auto-routing to departments
- Email notifications
- Database tracking
- One-click resolution

**Tech Stack:**
- Flask backend
- Twilio WhatsApp API
- PostgreSQL (Supabase)
- Resend email
- Deployed on Render

---

## 🔄 v1.1 - Enhanced Intelligence (2 weeks)

**Planned Features:**
- **ML Upgrade:** Replace rule-based with trained model
- **Multi-language:** Support Hindi, Tamil, Bengali
- **Voice Messages:** Transcribe and classify audio
- **Image Analysis:** Detect issues from photos
- **Response Templates:** Auto-suggest fixes

**Help Needed:**
- ML engineers for model training
- Translators for language support
- Testers with multilingual complaints

---

## 🎯 v2.0 - Complete Platform (2 months)

**Analytics Dashboard:**
- Real-time complaint statistics
- Department performance metrics
- Average resolution time tracking
- Category breakdown charts

**Mobile App:**
- Native app for departments
- Push notifications
- Offline support
- Photo uploads

**Advanced Features:**
- SMS fallback for email
- Recurring issue detection
- Predictive maintenance alerts
- Multi-campus support

**Contributors Wanted:**
- React/React Native developers
- UI/UX designers
- DevOps engineers
- Technical writers
